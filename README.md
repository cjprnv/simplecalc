# 🧮 SimpleCalc

A lightweight calculator application built in **Java**, covering core arithmetic operations with a clean, straightforward implementation.

## 📌 About

SimpleCalc is a beginner-friendly Java project that demonstrates fundamental programming concepts like object-oriented design, input handling, and arithmetic logic. It's a great starting point for understanding how console-based Java applications are structured.

## ✨ Features

- Addition, Subtraction, Multiplication, Division
- Clean console-based interface
- Handles basic user input
- Modular code structure under `src/`

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Java | Core language |
| JDK  | Compiler & runtime |

## 🚀 Getting Started

### Prerequisites
- Java JDK 8 or higher installed
- A terminal or IDE (IntelliJ / Eclipse / VS Code)

### Run Locally

```bash
# Clone the repository
git clone https://github.com/cjprnv/simplecalc.git
cd simplecalc

# Compile
javac src/*.java -d out/

# Run
java -cp out/ SimpleCalc
```
